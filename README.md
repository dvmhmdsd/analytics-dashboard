# Analytics-dashboard

Update the position and the size of charts by drag, drop and resize.

## Installation

1. Clone this repo.
2. Run the `index.html` file.

## Structure

```
.
├── index.html
├── js
│   ├── charts.js
│   ├── dom-handler.js
│   ├── main.js
│   └── store.js
├── README.md
└── style.css
```

## Dependencies

1. [Chart.js](https://www.chartjs.org/).
2. [Moveable](https://daybrush.com/moveable/).
